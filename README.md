# Product Manager App
A simple CRUD(Create, Read, Update, Delete) application built with native JavaScript to manage a product list.

## Tech Stack
- language: Vanilla JavaScript(ES6+)
- DOM Manipulation: DOM API
- Logic: Array methods(forEach, reduce)

## Features
- Dinamic Rendering: Automatically updates the table when data changes.
- Data Management: Interactive prompts for updating product details(name, count, price)
- Real-time Totals: Automatically calculates the total count and total cost of products using  reduce.
